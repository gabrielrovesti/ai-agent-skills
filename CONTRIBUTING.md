# Contributing to AI Agent Skills

Thanks for your interest in contributing.

## How to Contribute

1. Fork the repository
2. Create a branch for your change
3. Add or update skills in the `skills/` directory
4. Ensure docs are clear and consistent
5. Open a pull request with a clear summary

## Skill Contribution Checklist

Before submitting, confirm your skill:

- has a clear purpose
- includes usage examples
- defines input and output expectations
- documents limitations and edge cases
- follows existing naming and formatting conventions

## Pull Request Guidelines

- Keep changes focused and minimal
- Include documentation updates for behavioral changes
- Avoid unrelated refactors in the same PR

## Code of Conduct

Be respectful, constructive, and collaborative.
