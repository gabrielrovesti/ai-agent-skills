# Skills Directory

This folder contains reusable agent skills.

## Suggested Structure

For each skill, create a dedicated folder:

```text
skills/
└── <skill-name>/
    ├── README.md
    └── (optional supporting files)
```

## Skill README Expectations

Each skill README should explain:

- what the skill does
- when it should be used
- required inputs
- expected outputs
- concrete examples
- limitations or caveats
